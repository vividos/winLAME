Preset Selection Page

This is the preset selection pages. It only appears when the presets settings file (presets.xml) is installed. Also, presets for the specific output module must be available in the presets settings file to show this page.

The first entry "Custom settings" always brings you to the Output Module's configuration page. This is only recommended for experienced users, though. You can double-click on any entry to see what encoding parameters are used for this preset.

Preset List

Lists all available presets for the currently selected output module (from the output page). The description box shows the description for the currently selected preset. Double-clicking on one preset shows the settings parameter that are used for that preset.

Preset Description Box

Presents a description of the currently selected preset.
