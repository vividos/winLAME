CD Read Settings

Modern UI only

This is the CD read settings page. The page can be shown when clicking on the "CD Read Settings" button in the Modern UI.

User Interface

freedb Server

freedb.org is a service to retrieve CD information such as artist, track titles and so on. This setting configures the freedb server to use. Note that there is only one default entry "freedb.freedb.org (Random freedb server)". The best server is automatically chosen by freedb. You can type a specific server name here, if you want. Server name should be without http:// prefix. and without following slash.

Temporary CD Extraction Folder

When reading CDs to encode, a temporary folder is used to store the raw audio data. The folder is usually the %TEMP% folder of the system. Note that this setting is more relevant in Classic UI mode: The whole CD (about 700 MB) is read to this location before encoding the tracks. In Modern UI mode, only one track is stored here at most.

Store edited disc and track info in cdplayer.ini

In ancient Windows times there were CD player applications that store CD track info in an ini file stored in C:\Windows. When this check is set, winLAME stores the disc and track infos about a CD in this ini file, so that the other applications can re-use the infos.

Format various artists track

This text value determines how track filenames (and displayed track titles) are formatted. This gives you the possibility to adjust the generated filenames to your needs. The text can contain any characters, and the following words, placed between two % characters, are replaced with the track information accordingly.
KeywordDescription
albumInserts the album or disc name
artistInserts the artist of the track; may differ to albumartist
genreInserts the genre as text
titleInserts the track title
trackInserts the track number as digits
yearInserts the year where the album was released
albumartistInserts the album artist
lengthInserts the length of the track, in number of seconds
This setting is used when reading a CD with various artists (when the check box "Various Artists" is selected on the Input from CD page). The default text for this setting is: %track% - %album% - %artist% - %title%

Format album track

This setting is used when reading a CD with an album artist (check box "Various Artists" off). The default text for this setting is: %track% - %albumartist% - %album% - %title%
