Input CD Page

This is the input CD page where CD tracks can be selected to be extracted from an audio CD. Select the tracks to be used for encoding using the check boxes before each track. The shown CD disc and track infos are stored in the metadata (tags) when the track is encoded.

User Interface

CD disc drive

When there are multiple CD disc drives available, you can select the CD drive to use. The combobox is hidden when there is only one drive.

Track List

Shows the CD tracks on the current disc. When changing the CD in the drive, the tracks list is updated automatically. You can edit the track title, e.g. to fix spelling errors or to rename a track.

CD title

The CD disc title.

CD artist

The artist of all tracks. When there are several artist (checkbox Various Artists checked), the field can't be edited.

Various Artists

Indicates that there is no single artist for all tracks, but each track may have a different artist.

Year

Contains the year the CD was produced or recorded.

Genre

Contains the genre of the CD tracks. This is a combobox where preset genres can be selected. Own genre types can be typed in, too, but this is not supported in all output formats.

Buttons

Close

Closes the page.

Classic UI only

This button only appears in Classic UI mode. In Modern UI mode the next page is shown when clicking on the wizard navigation button.

Play

Plays the currently selected track. Note that this only works for CD players that are built into the PC and wired to the sound card. It doesn't work with external CD players.

Stop

Stops the currently playing track.

freedb.org

Queries the freedb.org CD database for disc and track infos and updates the current track list. When the CD isn't found in the database, a message box is shown.

Options

Shows the CD Audio Extraction options page.
