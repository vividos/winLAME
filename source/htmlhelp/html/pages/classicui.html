winLAME Classic User Interface

The Classic UI mode uses a wizard-dialog style appearance (looks like an Install Shield Wizard), to configure all details of encoding or decoding. The wizard makes it easy to click through the options and to encode the files when finished.

System Menu

Apart from the usual system menu items to resize and close the dialog window, there are some winLAME specific menu items.

Switch to Modern mode

Switches to Modern UI mode. The current settings and input files are lost.

Options

Shows the options dialog where you can set various options.

About winLAME

Shows the About dialog, showing all input and output modules and library versions used in winLAME.
