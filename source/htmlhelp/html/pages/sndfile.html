LibSndfile Settings Page

This is the LibSndfile settings page. LibSndfile is an external library that supports reading and writing many audio formats.

File Format

Lets you select the output file format that LibSndfile should use. There are many supported formats, and the list is updated dynamically as LibSndfile is upgraded between versions.

Sample Format

For every File Format there's a set of sample formats that the LibSndfile library can output. This lets you select the specific sample format that should be used. The list is updated dynamically when you select another file format. When a specific sample format is only supported for stereo or mono encoding, the format is marked "(Stereo)" or "(Mono)". The encoder fails on files with the wrong channel number then.
