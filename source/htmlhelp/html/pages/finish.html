Finish Page

The finish page is shown to check the files/audio tracks and settings and warns you about dangerous settings combinations. Closing the finish page creates the encoding and/or CD Audio extraction tasks and shows them in the main window of the Modern UI.

Settings warnings

The following warnings may appear, depending on the input files and settings chosen:
  • Warning! You are about to transcode audio file(s) from a lossy format to another. This may severely degrade sound quality!
  • Warning! You are about to overwrite the original input files with the encoded output files!
Please review carefully if you want to proceed. You may lose your original files that way.

Input tracks

Shows all input files or CD tracks that are to be encoded.

Output module

Shows the output module that was selected to be used.
